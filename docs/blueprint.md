# **App Name**: FleetView

## Core Features:

- Real-time Trip Visualization: Display vehicle positions on a map in real-time, simulating live tracking using event timestamps.
- Individual Trip Details: Show detailed metrics and progress for each of the 5 concurrent trips.
- Fleet-wide Metrics: Display overall fleet insights such as completed trips percentage and average trip duration.
- Data Simulation Controls: Provide playback controls (1x, 5x, 10x speed) to control the real-time simulation.
- Alert Aggregation and Summary: Use a tool to analyze real-time event data, aggregate and summarize alerts based on severity and frequency, providing actionable insights for fleet managers.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey trust and reliability.
- Background color: Very light gray (#F5F5F5), nearly white, to provide a clean and neutral backdrop, ensuring optimal contrast for UI elements.
- Accent color: A teal color (#009688), used to highlight key interactive elements and important data points, which ensures high contrast.
- Body and headline font: 'Inter' for a modern, neutral, and readable interface.
- Use clear, modern icons to represent vehicle status, event types, and control actions.
- Implement a responsive, modular layout to ensure the dashboard adapts well to different screen sizes.
- Subtle animations and transitions to provide visual feedback and enhance user engagement.
